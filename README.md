# CRUD-ALEJO
Programación orientada a objetos aplicada a un CRUD en Java, en este proyecto se trabaja con el manejo de clases como Persona, utilizando atributos y métodos para representar información de usuarios. Se emplean arreglos unidimensionales para almacenar múltiples objetos y estructuras de control como ciclos y condicionales para gestionar operaciones como registro, consulta, actualización y eliminación de datos.
Instrucciones de ejecución

Tener instalado Java (JDK)

Descargar los archivos del proyecto (Zip)

Descomprimir el archivo

Abrir la carpeta del proyecto en Visual Studio Code o cualquier IDE

Ubicarse en la carpeta donde están los archivos Main.java y Persona.java
